== 공통 사항

- 왼쪽 table을 통해 이동하시거나 cmd + f 또는 ctrl + f로 검색하셔서 찾을 수 있습니다.
- 파란 글씨로 된 링크를 클릭하시면 code(상수)를 새 창에서 보실 수 있습니다.

=== Domain

|===
| 환경 | Domain

| 개발서버
| `http://13.125.171.172`
| 개발서버 API 문서
| `http://13.125.171.172/docs/index.html`

| 운영서버
| 배포 예정
|===

=== 공통 에러 응답 - ErrorResponse

|===
| 필드명 | 설명
| `+status+`
| HTTP Status
| `+timestamp+`
| 응답 현재 서버 시각
| `+message+`
| 에러메시지
| `+errors+`
| 여러 필드가 잘못 되었을때의 필드 에러 정보

| `+path+`
| 요청 URL
|===
